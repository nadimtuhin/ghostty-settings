# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Reporting a vulnerability

Report privately via [GitHub Security Advisories](https://github.com/nadimtuhin/ghostty-settings/security/advisories/new).

Do not open a public issue for security vulnerabilities.

**Response time:** within 7 days.

**Scope:** shell injection in config parsing, installer download integrity issues.
